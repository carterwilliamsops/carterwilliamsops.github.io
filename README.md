This repository contains the source code for my personal portfolio website. It is designed to showcase my transition from Field Operations Engineer to Technical Product Manager, highlighting my work in Fleet Telematics, Hardware R&D, and Software Development.

## üöÄ About The Project

As a PM candidate with a background in heavy-duty EV repair and material science, I needed a portfolio that reflected two things:
1.  **Technical Competence:** Clean, modern code structure.
2.  **Product Thinking:** A user experience optimized for recruiters and hiring managers (scannable, responsive, and clear).

### Key Features
* **Single-File Architecture:** Lightweight and fast-loading with no build steps required.
* **Custom Design System:** Built on a customized `Slate` color palette to evoke a "Dark Mode" IDE/Engineering aesthetic.
* **Responsive UX:** Fully optimized for mobile and desktop viewing.
* **Case Study Integration:** dedicated sections for "Fleet Operations Optimization" (Software) and "Product Formulation Labs" (Hardware).
